# System Under Test (SUT)
## Description
This codebase represents the system being tested. A system is a group of features working together to provide a solution. Examples are power plants,drones and satelites, anything that can be represented as running software.

## Requirements
* Should be a group of interconnected containers with one container acting as an orchestrator.
* Should be able to consume external data from world-sims that need built to communicate data into and out of SUT.
